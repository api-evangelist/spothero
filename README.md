# SpotHero (spothero)

SpotHero is the leading digital parking marketplace in North America, offering a flexible parking API and developer toolkit that connects vehicles, drivers, and mobility apps with the largest network of off-street parking facilities. The platform enables navigation apps, rideshare services, connected cars, and enterprise fleets to seamlessly book and manage parking reservations. SpotHero was acquired by Uber in 2026 to power parking reservation experiences within the Uber app.

## APIs

### SpotHero Parking API

The SpotHero Parking API provides programmatic access to the largest network of off-street parking facilities in North America. Partners can search for available parking spots, check real-time availability, create and manage reservations, and access facility details including pricing, amenities, and directions. The API supports navigation apps, mobility platforms, connected car dashboards, event ticketing systems, and enterprise fleet management solutions. SpotHero covers 400+ cities with 8,000+ parking locations and has facilitated 100M+ parking reservations.

- **Human URL:** [https://spothero.com/developers](https://spothero.com/developers)
- **Base URL:** `https://api.spothero.com/v2`
